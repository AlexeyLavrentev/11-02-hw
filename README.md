# Домашнее задание к занятию «Кеширование Redis/memcached»

### Задание 1. Кеширование
Приведите примеры проблем, которые может решить кеширование.

### Ответ:
Кеширование применяется для решения следующих типовых проблем:
Часто запрашиваемые данные могут храниться в кеше, чтобы не выполнять один и тот же SQL-запрос много раз.
Результаты тяжёлых вычислений или сложных запросов можно сохранить в кеше и отдавать повторно без пересчёта.
Доступ к памяти значительно быстрее, чем к диску или внешним сервисам, что ускоряет ответы приложений.
Ответы от API сторонних сервисов можно кешировать, снижая количество запросов и риск блокировок.
Кеш помогает сгладить пики нагрузки, например во время акций или массовых входов пользователей.
